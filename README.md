# DSDBL

Data structures and database-related examples. This repository is primarily a Java learning project.

## What is in this repo

- Hadoop MapReduce example: `WordCount`
- Mapper, Reducer, and Driver implementation
- Java file: `hadoop.java`

## Purpose

Learn how to build simple MapReduce jobs with Hadoop and practice core Java data structure and file-processing patterns.

## Requirements

- Java 8+
- Hadoop or compatible MapReduce runtime if you want to execute the job

## How to use

Review `hadoop.java` for the MapReduce job structure. Compile and run with Hadoop classpath if you want to execute it locally.
